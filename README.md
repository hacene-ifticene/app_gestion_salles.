Une description claire et concise de l’atelier:

Cet atelier consiste à concevoir et développer une application python de gestion des salles
avec une interface graphique (gui)
L'application vas nous permettre d'effectuer les opération crud
(ajouter,modifier,supp,rechercher et afficher)
sur des salles stockées dans une base des données MySQL.
Ce projet est structuré egalement selon une architecture en couche tel que(data,models,service et views)
afin de s.parer l'acces aux donnés, la logique métier et l'interface utilisateur.
Il inclut égalment l'utilisation de git et GitHub et ca c'est trés important pour le suivi des version
avec des commits réguliers reflétant les différentes étapes du développement de ce dernier.
